smarthome

"SMARTHOME" merupakan project Internet of Things dimana aplikasi ini dapat mengendalikan perangkat elektronik pada prototype rumah. Perangkat yang dikendalikan yaitu dua buah lampu led, satu buah mini doorlock pada pintu, dua buah servo yang digunakan untuk membuka jendela dan pagar pada prototype rumah. Project ini terdiri dari website, aplikasi android dengan memanfaatkan webview, dan sebuah prototype rumah yang sudah dilengkapi perangkat elektronik untuk dikendalikan.

Pada project ini menggunakan framework CodeIgniter dan memanfaatkan JavaScript serta JQuery, saya mengerjakan program mikrokontroller arduino setiap perangkat yang akan dikendalikan dan menghubungkannya pada website yang mengendalikannya menggunakan PHP dan Json untuk transfer data antar platform yaitu arduino dan website. Anda dapat mengunjungi websitenya di smarthome.ourlabpro.com

Coklit

"COKLIT (Pencocokan Data dan Penelitian) Data Kinerja PDAM Berbasis WEB" merupakan project saya ketika magang bersama dua rekan saya lainnya di Workshop PT. Geoinfo Teknologi Banjarmasin. Aplikasi ini bertujuan untuk membandingkan hasil akhir dari kinerja suatu pdam dengan nilai akhir hasil perhitungan menggunakan rumus BPPSPAM.

Pada project yang kami kerjakan bersama ini menggunakan framework CodeIgniter, saya mengerjakan halaman pencocokan yang menampilkan perbandingan hasil dari data yang sudah ada dengan data hasil perhitungan sistem. Apakah cocok atau tidak menggunakan PHP. saya juga mengerjakan halaman kinerja yang menunjukkan range nilai kinerja PDAM menggunakan PHP dan JavaScript.

prototype sistem pendeteksi kebakaran hotel dengan IoT berbasis Nodemcu

"Prototype Sistem Pendeteksi Kebakaran Hotel dengan IoT Berbasis Nodemcu" adalah project IoT dimana pengguna dapat memantau keeadaan suhu dan kebocoran gas yang memungkinkan timbulnya kebakaran serta dapat mengontrol lampu melalui website. Project ini menggunakan mikrokontroller NodeMcu, sensor suhu dan sensor gas.
Pada project ini saya membuat website dengan PHP Native yang dikombinasikan dengan bootstrap dan adminLTE. Untuk pengendalian lampu saya menggunakan json dalam transfer data dari mikrokontroller ke web.

Sistem monitoring bak sampah

"Sistem monitoring bak sampah" adalah project IoT yang dapat memonitoring kondisi bak sampah dari mana saja dan kapan saja. Memanfaatkan mikrokontroller nodemcu serta beberapa perangkat hardware lainnya dalam pembuatan bak sampahnya.
Pada project ini saya membuat CRUD sederhana yang ditampilakn dengan dataTables. Disini saya menggunakan framework CodeIgniter yang dikombinasikan dengan adminLTE dan bootstrap dalam pembuatan websitenya.

Sistem monitoring pagar otomatis

"Sistem monitoring pagar otomatis" adalah project IoT dimana pengguna dapat mengendalikan pagar (membuka ataupun menutup) melalui website dan sms yang pembuatan prototypenya memanfaatkan mikrokontroller arduino dan modul sms gateway.
Pada project ini saya membuat CRUD sederhana yang ditampilakn dengan dataTables. Disini saya menggunakan PHP Native yang dikombinasikan dengan adminLTE dan bootstrap dalam pembuatan websitenya.

Sistem monitoring pelanggaran lalu lintas

"Sistem monitoring pelanggaran lalu lintas" adalah Project IoT yang membantu memantau saat terjadinya pelanggaran lampu lalu lintas. Menggunakan mikrokontroller NODEMCU dalam mengendalikan kamera pengawas yang disambungkan dengan aplikasi delphi dan hasilnya akan terlihat pada website.
Pada project ini saya bertugas dalam membuat website CRUD sederhana yang ditampilkan dengan dataTables. Saya menggunakan framwork CodeIgniter dalam pembuatannya dan menggunakan library DomPdf untuk menjadikan hasil report dalam pdf.
